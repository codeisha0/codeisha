# Simple Home Page

A simple responsive home page with a contact form built using HTML and CSS.

## Features

- Responsive layout
- Navigation bar
- Hero section
- Contact form
- Clean and modern design

## Technologies

- HTML5
- CSS3

## Run Locally

1. Clone the repository

```bash
git clone https://github.com/your-username/simple-homepage.git
```

2. Open `index.html` in your browser.
